# Citations

## AI Courtesy

**We did not use AI to write any code or give us ideas. Our software makes API calls to an AI Model from Google or OpenAI to summarize webpages or return a function from a user's prompt.**

Briefly has two main features, summarization and an AI Agent.

- Briefly can summarize a webpage and read the summary aloud.

- Briefly also has an AI Agent that can navigate through the web and answer questions like Siri.
